Membangun Hybrid Android App dengan Ionic dan Cordova

2015-03-25
Mobile App

Ionic Framework adalah front-end SDK untuk membangun hybrid mobile app dengan HTML5.

Apache Cordova adalah platform untuk membangun native mobile app menggunakan HTML, CSS, dan JavaScript.

Instalasi

  • Unduh dan jalankan script ubuntu-ionic-installer —> sudo ubuntu-ionic-installer.sh

Proses

  • Buat app —> ionic start namaApp blank

Opsional > daftar ke Ionic > login > unggah app > install Ionic View untuk live testing.

  • Masuk ke direktori app & instal ngCordova —> cd namaApp && bower install ngCordova

Opsional > hanya jika UNMET DEPENDENCY —> sudo npm install gulp gulp-concat gulp-minify-css gulp-rename gulp-sass

  • Kustomisasikan app dalam direktori www

  • Tentukan platform —> ionic platform add android

  • Bangun app —> ionic build android

  • Emulator —> ionic emulate android

Bagikan
  • Apache Cordova
  • Ionic Framework
