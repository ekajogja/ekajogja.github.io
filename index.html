Membuat virtual environment untuk Node.js dengan nodeenv

nodeenv adalah tool untuk membuat virtual environment Node.js.

  • dependensi —> python, pip

  • install —> $ sudo pip install nodeenv

  • create —> $ nodeenv --node=0.12.0 --prebuilt namaEnv

  • activate —> $ . namaEnv/bin/activate

  • deactivate —> (env) $ deactivate_node
