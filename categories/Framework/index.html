Instalasi dan Membuat Test App dengan Express.js

2015-03-24
Framework

Express adalah web framework minimalis berbasis Node.

Lihat contoh aplikasi yang dibangun dengan Express disini.

  • Buat virtual environment sesuai panduan 00_nodeenv.md.
    Aktifkan dan masuk ke direktori tsb.

  • Buat direktori aplikasi dan masuki —> mkdir namaApp && cd namaApp

  • Buat file package.json yang memuat identitas dan dependensi aplikasi —> npm init
    Jawab semua prompt untuk mengisi identitas aplikasi, atau cukup klik Enter untuk default value.
    Is this ok? (yes) yes

  • Instal Express —> sudo npm install express --save

  • Periksa bahwa Express telah tersimpan (otomatis) dalam daftar dependensi package.json. Kurang lebih begini:

  • Instal tool generator aplikasi —> sudo npm install express-generator

  • Bangun aplikasi —> express namaApp
    Otomatis membuat direktori anak “namaApp” di dalam direktori induk “namaApp”.

  • Masuk ke direktori anak dan instal dependensi —> cd namaApp && sudo npm install

  • Jalankan aplikasi —> DEBUG=namaApp ./bin/www
    Buka http://localhost:3000/ di browser untuk melihat pesan “Welcome to Express”.

Temukan referensi lanjut di sini.
